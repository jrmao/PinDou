# 拼豆图案设计工具 - 设计文档

**日期：** 2026-05-02

**状态：** 已批准

---

## 1. 项目概述

- **项目名称**：PinDou（拼豆图案设计工具）
- **项目类型**：本地单文件网页应用
- **核心功能**：像素画编辑器 + 图片转图案 + 图案模板库
- **目标用户**：拼豆爱好者、手工达人

---

## 2. 技术方案

### 2.1 技术栈
- **纯原生 HTML + CSS + JavaScript**：无框架依赖
- **HTML5 Canvas 2D**：像素画渲染
- **单文件部署**：所有代码内联到一个 `.html` 文件，双击即可运行

### 2.2 技术要点
- ES6+ 模块化代码组织（虽为单文件，但代码内部按模块划分）
- 事件委托机制减少事件监听器
- 防抖/节流优化频繁操作性能

---

## 3. 功能模块

### 3.1 像素画编辑器

| 功能 | 说明 |
|------|------|
| 画布尺寸 | 最小 10×10，最大 200×200，支持自定义 |
| 绘制工具 | 铅笔（单点绘制）、橡皮擦、填色桶、矩形工具、直线工具 |
| 操作 | 撤销/重做（Ctrl+Z/Y）、缩放（鼠标滚轮）、平移（拖拽） |
| 颜色 | 预设 Mard 291 色基础盘 + 用户自定义颜色管理 |

### 3.2 图片转图案

| 功能 | 说明 |
|------|------|
| 导入方式 | 点击上传或拖拽（PNG/JPG） |
| 参数设置 | 网格尺寸（8×8 到 100×100）、颜色匹配算法 |
| 转换模式 | 默认自动转换，过程中支持手动干预 |
| 预览 | 转换前/后对比视图 |
| 编辑 | 转换后进入像素编辑器微调 |

### 3.3 图案模板库

| 功能 | 说明 |
|------|------|
| 内置模板 | 10-20 个基础图案（卡通、字母、emoji） |
| 用户模板 | 用户可保存当前作品为自定义模板 |
| 展示 | 缩略图网格布局，点击选中 |

---

## 4. 颜色体系

### 4.1 默认预设：Mard 291 色

| 系列 | 色调 | 数量 |
|------|------|------|
| A | 黄橙色 | ~26 |
| B | 绿色 | ~32 |
| C | 蓝青色 | ~29 |
| D | 紫蓝色 | ~26 |
| E | 粉红色 | ~24 |
| F | 红色 | ~25 |
| G | 橙棕色 | ~21 |
| H | 黑白灰 | ~23 |
| M | 莫兰迪色 | ~15 |
| P/Q/R/T/Y/Z | 扩展系列 | ~70+ |

**总计：291 色**

### 4.2 颜色属性

```json
{
  "num": "A1",
  "type": "normal",
  "color1": "rgb(250,245,205)"
}
```

### 4.3 用户自定义
- 可添加/删除/编辑颜色
- 支持按色号或 HEX 搜索

## 5. 数据存储

| 功能 | 说明 |
|------|------|
| 自动保存 | 每 30 秒自动保存到 LocalStorage |
| 手动保存 | 可给图案起名保存 |
| 导出 | 支持导出为 PNG 图片 |
| 加载 | 可从 LocalStorage 加载历史作品 |
